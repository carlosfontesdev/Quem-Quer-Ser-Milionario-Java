# 💰 Jogo: Quem Quer Ser Milionário? (Java Swing)

Este é um projeto desenvolvido em **Java** que recria a experiência do famoso concurso televisivo "Quem Quer Ser Milionário". O jogo utiliza uma interface gráfica intuitiva e mecânicas clássicas de perguntas e respostas.

---

## 📸 Demonstração do Jogo

Para tornar a experiência mais visual, aqui estão as principais telas do sistema:

### 🏠 Menu Principal
Onde o jogador faz o seu registo para iniciar o desafio.
![Início](Inicio.JPG)

### 🎮 Interface de Jogo
Sistema de perguntas com feedback visual imediato ao acertar.
![Acerto](PerguntaAcertada.JPG)

### ❌ Fim de Jogo
Ecrã que exibe o prémio final acumulado pelo jogador.
![Erro](PerguntaErrada.JPG)

---

## ✨ Funcionalidades

* **Registo de Jogador:** Nome e data de nascimento.
* **Progressão de Prémios:** Escala de valores que aumenta a cada acerto (de 100€ a 1000€).
* **Sistema de Ajudas:**
    * **50/50:** Remove duas opções erradas para facilitar a escolha.
    * **Mudar Pergunta:** Permite trocar a questão atual por uma nova.
* **Interface Gráfica (GUI):** Criada com Java Swing para uma experiência amigável.

## 🚀 Como Executar o Projeto

1.  Faz o download deste repositório como um ficheiro `.zip` ou clona-o.
2.  Abre o **NetBeans IDE**.
3.  Vai a `File > Open Project` e seleciona a pasta do projeto.
4.  Clica no botão **Run (F6)**.
5.  *Alternativamente*, podes tentar executar o ficheiro `jogar.bat` se estiveres em Windows.
